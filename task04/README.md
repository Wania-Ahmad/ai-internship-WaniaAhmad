## Part C – Mini Project (Titanic Dataset)
- **Dataset:** `titanic.csv`
- **Cleaning:** Filled missing Age values with median, Embarked with mode.
- **Summary Statistics:** Used `describe()` to explore dataset.
- **Findings:**
  - Overall survival rate: ~38%
  - Survival by class: 1st class ~63%, 2nd class ~47%, 3rd class ~24%
  - Survival by gender: Female ~74%, Male ~19%
- **Conclusion:** Survival strongly depended on **class** and **gender**.
